# バーチャル支援員 AI 企画書

## コンセプト

**「いつでも話せる、あなたを覚えている支援員」**

毎朝の会話から面接練習まで、利用者に寄り添うAI支援員。
会話を重ねるほど、その人のことを理解していく。

---

## やりたいこと

### 1. 毎朝のチェックイン

> 「おはよう！昨日映画観るって言ってたけど、観れた？」

- 朝15分、AIと雑談
- 前日までの会話を覚えていて、続きの話ができる
- 体調確認、気分の変化を記録
- 支援員さんは後からログを確認できる

---

### 2. 面接練習

> 「今日は志望動機の練習してみない？」

- 定型質問を音声で出題
- 回答を録画・録音
- 後から分析結果が届く（赤ペン先生方式）

**分析する項目**
- 視線（カメラ見てるか）
- 表情（笑顔、緊張度）
- 話し方（スピード、声の大きさ）
- 内容（論理性、具体性）

---

### 3. 夜間バッチ処理

> 毎晩、翌日の準備をAIがしておく

- 過去の会話履歴を振り返り
- 翌朝の話題を自動生成
- 面接練習の分析レポート作成
- 「結果が届きました！」で通知

---

### 4. パーソナリティの蓄積

> 会話するほど、その人を理解する

- 趣味、興味、得意なこと
- 苦手なこと、配慮事項
- 体調の傾向
- 会話から職務経歴も自動蓄積

---

### 5. 支援員ダッシュボード

> 支援員さんが利用者の状態を把握できる

- 毎日の会話ログ
- 気分・体調の推移
- 面接練習の進捗
- AIが気づいた変化のアラート

---

## ユーザー体験イメージ

```
━━━ 朝 ━━━

　　そういえば、面接練習の結果出てるよ。見てみる？

利用者：見る！

AI：視線キープ率が68%で、前回より上がってる！
　　もう少しカメラを見る練習すると、もっと良くなりそう。
　　今日も1問だけ練習してみない？

利用者：やる

AI：じゃあ今日は「長所と短所」いってみよう。
　　準備できたら教えてね。

━━━ 夜 ━━━

[夜間バッチ実行]
- 本日の会話分析
- 面接練習の採点
- 明日の話題準備
  → 「肉じゃがの話の続き」
  → 「長所短所の練習結果フィードバック」
  → 「次は自己PRの練習を提案」
```

## 技術構成

### 使用API・サービス

| 用途 | サービス | 備考 |
|------|----------|------|
| 認証 | Microsoft Teams | 既存の基盤を利用 |
| 音声認識 | Deepgram | ストリーミング対応 |
| LLM（会話・分析） | Gemini 1.5 Flash | コスパ◎ |
| LLM（フィードバック） | Claude Haiku | 日本語品質◎ |
| LLM（高速会話） | Groq | リアルタイム会話用 |
| 表情・視線分析 | MediaPipe | 無料、ローカル処理 |
| 音声合成 | VOICEVOX | 無料、日本語自然 |
| データベース | BigQuery | 会話履歴、分析結果 |
| 動画保存 | Cloud Storage | 面接練習の録画 |
| バッチ処理 | Cloud Run + n8n | 夜間処理 |

### 高速化オプション

リアルタイム会話の体感速度を上げる方法：

| 方式 | 体感遅延 | コスト | 説明 |
|------|----------|--------|------|
| 相槌即返し | 0.5秒 | 安い | 「うん」を即返し、裏でLLM処理 |
| Groq | 0.8秒 | 安い | 超高速LLM |
| OpenAI Realtime | 0.3秒 | 高い | 音声→音声の直接処理 |

## 質問セット（面接練習）

### 基本質問
1. 自己紹介をお願いします
2. 志望動機を教えてください
3. これまでの経験で頑張ったことは？
4. 長所と短所を教えてください
5. 5年後どうなっていたいですか

### 就労支援向け
1. 体調管理で気をつけていることは？
2. 苦手なことや配慮してほしいことは？
3. 得意な作業や好きな作業は？
4. 困ったときはどうしますか？
5. 働く上で大切にしたいことは？

## 検討事項

- **プライバシー**：動画・会話データの保管期間、支援員への共有範囲
- **アクセシビリティ**：視覚・聴覚障がいへの対応、シンプルなUI
- **同意取得**：利用者への説明と同意プロセス
