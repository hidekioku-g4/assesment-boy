# インタビュープラクティス MVP 切り出し

## 目的
- 既存の文字起こし基盤を使い、面接練習の最小体験を提供する
- まずは「質問 → 回答 → テキストフィードバック」までを成立させる

## MVPでやること（In Scope）
- 質問セットの選択（基本／就労支援向け）
- 1問ずつの質問提示（前へ／次へ）
- 回答テキストの取得（既存のクリーン済み文字起こしを取り込む）
- 回答の手直し（テキスト編集）
- LLMによるフィードバック生成（テキストのみ）
- データ永続化は行わない（無保存モードに準拠）

## 今回はやらない（Out of Scope）
- 視線・表情・声量などのマルチモーダル分析
- 動画／音声の保存
- 連続セッションの履歴・ダッシュボード
- パーソナリティの自動蓄積（長期記憶）
- 音声合成（読み上げ）

## MVP後の拡張候補
- 質問セットのカスタム編集・保存
- フィードバックの採点ルーブリック化（項目別スコア）
- BigQueryへの結果保存と支援員ダッシュボード表示
- 朝のチェックイン機能（短い雑談＋記録）
- 夜間バッチ（翌日話題／レポート生成）

## 進め方（最短の動線）
1. 文字起こし開始 → 話す
2. クリーン実行
3. 面接練習パネルで「クリーン結果を取り込む」
4. フィードバック生成
